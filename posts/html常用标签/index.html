HTML常见标签用法

a标签

<a href="http://baidu.com" target="_blank" download rel=noopener>
</a>

/href链接域名；target是指链接网页打开的位置，有打开一个网页_blank，当前网页_self，头部网页_top，父级网页_parent。/ 

<a href="javascript:alert(哈哈哈);">点我</a>

/点击“点我”就会弹出“哈哈哈”/

<a href="mailto:1804815164@qq.com">发邮件给我</a>

/点击“发邮件给我”就会进入邮箱写邮件/

<a href="tel:17784414880">打电话给我</a>

/点击“打电话给我”就会转入到拨电话界面/

iframe标签

<iframe src="http://baidu.com" iframe.border="0"></iframe>

/在当前页面直接连接显示百度页面；也可以连接另外一个文件/

table标签

<table>
    <thead>
        <tr>
            <th></th>
            <td>小明</td>
            <td>小张</td>
            <td>小李</td>
        </tr>
    </thead>
    <tbody>
        <tr>
            <th>语文</th>
            <td>100</td>
            <td>100</td>
            <td>100</td>
        </tr>
        <tr>
            <th>数学</th>
            <td>100</td>
            <td>100</td>
            <td>100</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <th>总计</th>
            <td>200</td>
            <td>200</td>
            <td>200</td>
        </tr>
    </tfoot>
<table>

/复制这段代码打开网页查看/ 

table{table-layout:auto/fixed;}

/改变表格样式，auto是根据内容自动调整宽度；fixed是等宽/

img标签

<img id="xxx" src="" alt="" width="" height=""/>

/id是名称；src是source的缩写，get到一张图片链接，alt是alter的缩写，意为“改变，更改”，填写要更改的内容。/ 

<script>
    xxx.onload=function(){console.log(图片加载成功)};
    xxx.onerror=function(){console.log(图片加载失败);xxx.src="/404.jpg"};
</script>

/onload显示加载成功，onerror加载失败后get到404图片。/ 

<style>
    *{margin:0; padding:0; box-sizing:border-box;}
    img{max-width:100%;}
</style>

/创建响应式网站就是img{max-width:100%;}/

form标签

<form action="/xxx" method="post" autocomplete="on">
    用户名<input name="username" type="text">
    密码<input type="password">
    <input type="submit">

/action连接到的页面；method跳转方法get/post；autocomplete一般与name连用，输入文本是会弹出建议/

input标签

<input type="submit">
<button type="submit">提交</button>

/两者都可以提交内容，但button中间可以加其他标签/ 

<input name="gender" type="radio">男
<input name="gender" type="radio">女

/相同name为一组，radio表示单选/ 

<input name="hobby" type="checkbox">唱
<input name="hobby" type="checkbox">跳
<input name="hobby" type="checkbox">rap

/check表多选/ 

<input type="file" multiple>

/multiple表上传多个文件，去掉只能上传单个文件/ 

<input type="hidden">
<input type="color">

/hidden是隐藏文本框，color是文本框填充颜色/ 

<textarea style="resize:none;width:30%;height:100%">
</textarea>

/textarea可缩放文本框，resize:none使其不可动拖动/ 

<input type="text" required>

/提示文本框必须填写内容/ 

<select>
    <option value="">-请选择-</option>
    <option value="1">星期一</option>
    <option value="2">星期二</option>
    <option value="3">星期三</option>
</select>

/下拉选择型文本框/

注意

  • 一般不监听input的click事件
  • form里面的input要有name
  • form里面要有type=“submit”才能触发文本框
